# Azure Generative AI App with Custom Data (RAG)

## Overview

This project is based on a hands-on lab completed while preparing for the **Azure AI Engineer professional certification** using Microsoft Learn.

The lab demonstrates how to build a **Generative AI application that answers questions using custom data** through a Retrieval-Augmented Generation (RAG) approach.

## What I Learned

* Deploying and using Azure OpenAI models
* Connecting AI models with custom data
* Implementing Retrieval-Augmented Generation (RAG)
* Building a simple AI-powered question answering system

## Technologies Used

* Azure OpenAI
* Azure AI Search
* Python
* Vector embeddings
* Retrieval-Augmented Generation (RAG)

## Purpose

The purpose of this project is to document my hands-on experience with Azure AI tools while learning how to develop Generative AI applications using real-world data.

## Note

This project was completed as part of a hands-on lab during my preparation for the **Azure AI Engineer certification**.
